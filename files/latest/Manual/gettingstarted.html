DOOSS User Documentation


Introduction

DOOSS is a tool for visualising RNA and DNA secondary structures.

Platforms supported

DOOSS has been developed for use on Windows, but should run on Linux and Mac (not yet tested).

Requirements

Jave Runtime Environment (JRE) version 1.6.0 or greater is required to run DOOSS and at least 1GB of memory is recommended.

Installing DOOSS

DOOSS is designed to be portable and therefore does not come with an installer, in order to use it in on your computer, simply extract the DOOSS.zip to the location of your choice. This is a cross-platform installation.

Running DOOSS

In order to run DOOSS navigate to you DOOSS directory and launch the 'DOOSS.jar' file. If you are using Windows you can double-click the 'DOOSS.exe' executable.
